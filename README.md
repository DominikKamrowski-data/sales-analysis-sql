# Sales Analysis SQL Project

SQL project analyzing sales revenue by product, region and order size using SQLite.

## 📌 Overview
This project analyzes sales data using SQL (SQLite) to identify key business insights and trends.

## Business Context

The goal of this project was to analyze sales data and identify which products and regions generate the highest revenue. The analysis also compares large and small orders to support basic business decision-making.

## 🛠 Tools
- SQL (SQLite)

## 📈 Analysis Performed
- Revenue by product  
- Revenue by region  
- Large vs small orders analysis  
- Top product in each region  

## 🔍 Key Insights
- Whisky generates the highest revenue despite lower sales volume compared to other products  
- Warsaw and Gdańsk are the strongest regions  
- Small orders generate higher total revenue than large ones  

## 📂 Files
- data.sql → database structure and data  
- analysis.sql → SQL queries used for analysis
